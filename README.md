# Laboratory-reservation实验室预约

#### 系统介绍

实验室预约管理系统是一套集实验室资源管理、智能预约、安全准入、门禁联动与移动端服务于一体的综合性实验室预约管理平台。系统以图形化方式直观展示全校可预约实验室资源，用户可便捷查看实验室详情，包括名称、地址、负责人、实景图片及仪器设备清单。支持工位预约或者整屋预约，自动校验课表冲突与实验室最大容纳人数限制；可与实验室安全准入联动，要求先完成安全培训与考试后方可预约。

也可与门禁系统联动，审核通过后，自动为用户开通对应时段的实验室门禁权限，支持人脸识别、刷卡或二维码等多种开门方式，确保权限精准、安全可控。

用户可在“我的预约”中查看所有预约记录，并按规则取消预约；管理员则可通过“预约汇总”模块全面掌握实验室使用情况，支持按实验室、单位、时间段等多维度筛选，并一键导出Excel报表，便于统计与决策。

系统全面支持移动端操作，真正实现“随时随地、高效便捷”的实验室智慧管理，提升资源利用率与实验安全水平。

#### English version introduction

#### System Introduction

The laboratory reservation management system is a comprehensive laboratory reservation management platform that integrates laboratory resource management, intelligent reservation, secure access, access control linkage, and mobile services. The system visually displays laboratory resources that can be reserved throughout the school in a graphical manner. Users can easily view laboratory details, including names, addresses, responsible persons, real-life pictures, and equipment lists. Support workstation reservation or whole house reservation, automatically verify schedule conflicts and laboratory maximum capacity limit; It can be linked with laboratory safety admission, requiring completion of safety training and exams before making an appointment.

It can also be linked with the access control system. After approval, the laboratory access control permission for the corresponding time period will be automatically opened for the user. It supports multiple door opening methods such as facial recognition, card swiping, or QR code, ensuring accurate, secure, and controllable permissions.

Users can view all appointment records in "My Appointment" and cancel appointments according to the rules; Administrators can comprehensively grasp the usage of the laboratory through the "Appointment Summary" module, which supports multi-dimensional filtering by laboratory, unit, time period, etc., and can export Excel reports with one click for easy statistics and decision-making.

The system fully supports mobile operations, truly realizing "anytime, anywhere, efficient and convenient" laboratory intelligent management, improving resource utilization and experimental safety level.
